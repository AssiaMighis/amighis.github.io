# Assia Mighis — Personal site (GitHub Pages)

This is a static one‑page site for GitHub Pages.

## Deploy on GitHub Pages (user site)

1. Create a **public** repository named **`<username>.github.io`** (replace with your GitHub username).
2. Upload `index.html` and `style.css` at the repository root.
3. Commit & push. Your site will be served at **https://<username>.github.io/**.

## Local preview
Open `index.html` in your browser, or run a simple server:

```bash
python -m http.server 8000
```

## Customize
- Edit text in `index.html`
- Adjust styles in `style.css`
- Add images in `assets/` and reference them from HTML
